# Markdown作成ガイドライン

(linterは別として以下を守る)

## 全般的なルール

- 箇条書きを中心に構造化する

## 改行に関するルール

- 基本的に1行1文。異なる文を書く時は特別な理由がない限り改行する
- 横に長過ぎる場合は同じ文内でも改行する
    - utf-8日本語換算60文字以上, 英数字換算100文字以上が目安
- 見出しと本文の間は常に1行改行を入れる
- 見出しと見出しの間は常に1行改行を入れる
